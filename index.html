Cara Pemesanan Ruangan

  1. Login ke Sistem
    Masuk menggunakan akun kampus Anda.
  2. Pilih Menu "Buat Pemesanan"
    Klik tombol Buat Pemesanan pada menu Aksi Cepat.
  3. Isi Formulir Pemesanan
    Lengkapi data pemesan, pilih ruangan, tanggal, waktu, dan keterangan kegiatan.
  4. Kirim Pemesanan
    Klik tombol Kirim Pemesanan untuk mengajukan permintaan.
  5. Cek Status Pemesanan
    Lihat status pemesanan Anda di menu Jadwal Saya atau Kelola Pemesanan.
  6. Cetak Bukti
    Setelah pemesanan dikonfirmasi, cetak bukti pemesanan melalui menu Cetak Bukti.
Jika mengalami kendala, hubungi admin kampus melalui kontak yang tersedia di halaman ini.
